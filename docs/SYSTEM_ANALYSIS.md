# Fitur Imageboard Saat Ini

Berikut ini adalah daftar fitur yang sudah aktif dan bisa langsung digunakan:

## 1. Board dan Konten

- **Banyak Board**: Mendukung banyak kategori board (seperti /a/, /b/, dsb) dengan kode unik masing-masing.
- **Thread dan Balasan**: Sistem diskusi standar dengan nomor post unik untuk setiap kiriman.
- **Bumping**: Thread yang ada balasan baru otomatis naik ke urutan paling atas.
- **Katalog**: Tampilan grid untuk melihat semua thread dalam satu board sekaligus.
- **Pencarian**: Fitur untuk mencari thread berdasarkan judul atau isi di dalam board tertentu.

## 2. Media dan Interaksi

- **Upload Gambar**: Bisa upload gambar di setiap postingan.
- **Info Gambar**: Menampilkan detail gambar seperti resolusi, ukuran file, dan nama asli file.
- **Pratinjau Gambar**: Gambar bisa diperbesar langsung. Konten NSFW atau Spoiler otomatis diburamkan sampai diklik oleh user.
- **Video Embed**: Link YouTube otomatis berubah jadi pemutar video di dalam post.
- **Quick Reply**: Kotak balasan melayang supaya bisa membalas tanpa harus scroll atau pindah halaman.

## 3. Identitas dan Anonimitas

- **Anonim**: User bisa posting tanpa harus login atau memasukkan nama (defaultnya "Awanama").
- **Tripcode**: Cara untuk punya identitas tetap tanpa akun, menggunakan hash dari password.
- **Poster ID**: ID unik per thread untuk membedakan orang yang memposting, tapi tetap menjaga identitas asli.

## 4. Format Pesan dan Navigasi

- **Kutipan (Quoting)**: Bisa mengutip post lain menggunakan nomor post yang otomatis jadi link.
- **Pratinjau Kutipan**: Muncul pop-up isi post kalau kursor diarahkan ke nomor post yang dikutip.
- **Backlinks**: Menampilkan daftar post yang mengutip post tersebut di bagian bawah.
- **Greentext**: Baris yang diawali tanda ">" otomatis berubah warna jadi hijau.
- **Spoiler Teks**: Teks bisa disembunyikan dan baru muncul kalau diklik.
- **Smooth Scroll**: Halaman akan otomatis bergeser ke post yang dituju saat link kutipan diklik.

## 5. Moderasi dan Keamanan

- **Lapor Post**: Fitur untuk melaporkan kiriman yang melanggar aturan.
- **Panel Moderasi**: Halaman khusus untuk admin atau moderator mengelola konten.
- **Kontrol Thread**: Bisa menyematkan (pin), mengunci (lock), atau menghapus post.
- **Sistem Ban**: Bisa memblokir akses berdasarkan IP dengan alasan dan durasi tertentu.
- **Verifikasi Umur**: Dialog verifikasi umur untuk pengunjung baru.

## 6. Antarmuka dan UX

- **Tema**: Pilihan mode gelap (Dark Mode) dan terang (Light Mode).
- **Responsif**: Tampilan menyesuaikan dengan baik di HP maupun komputer.
- **Navigasi Atas**: Bar navigasi untuk pindah antar board dengan cepat.
